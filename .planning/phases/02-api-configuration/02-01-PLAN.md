<task type="auto">
  <name>Task 2: Implement storage handlers in main.ts</name>
  <files>src/main.ts</files>
  <action>
Add switch cases in the `handleUIRequest` function to handle storage operations:

1. Handle GET_API_KEY:
   ```typescript
   case 'GET_API_KEY': {
     const apiKey = await figma.clientStorage.getAsync('anthropic_api_key');
     postToUI({
       type: 'RESPONSE',
       correlationId,
       payload: { apiKey: apiKey ?? null }
     });
     break;
   }
   ```

2. Handle SET_API_KEY:
   ```typescript
   case 'SET_API_KEY': {
     const { key } = payload as { key: string };
     await figma.clientStorage.setAsync('anthropic_api_key', key);
     postToUI({
       type: 'RESPONSE',
       correlationId,
       payload: { success: true }
     });
     break;
   }
   ```

3. Handle CLEAR_API_KEY:
   ```typescript
   case 'CLEAR_API_KEY': {
     await figma.clientStorage.deleteAsync('anthropic_api_key');
     postToUI({
       type: 'RESPONSE',
       correlationId,
       payload: { success: true }
     });
     break;
   }
   ```

4. The `handleUIRequest` function needs to become async to support await. Update the signature:
   ```typescript
   async function handleUIRequest(msg: UIMessage & { type: 'REQUEST' }): Promise<void>
   ```

Use storage key name: `anthropic_api_key` (matches research recommendation).
