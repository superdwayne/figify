  <name>Task 1: Add confidence thresholds to LayoutStructurer</name>
  <files>src/main/generator/layoutStructurer.ts</files>
  <action>
Update LayoutStructurer to be more conservative about creating virtual containers:

1. Increase MIN_GROUP_SIZE from 2 to 3 - require at least 3 elements to form a meaningful group (rows/columns with 2 elements are often false positives)

2. Increase coverage ratio threshold from 0.5 to 0.7 - only create containers when pattern covers at least 70% of sibling elements

3. Add spacing consistency check for rows/columns - only group elements if their spacing is reasonably uniform (max 2x variation in gap sizes)

4. Add size consistency check - elements in a row should have similar heights (within 50%), elements in a column should have similar widths (within 50%)

5. For grid detection, require at least 2x2 (4 elements) minimum AND consistent row counts (no partial grids)

6. Add a `shouldCreateContainer()` method that checks:
   - Group has >= MIN_GROUP_SIZE members
   - Coverage ratio >= MIN_COVERAGE_RATIO (0.7)
   - Spacing is uniform (if applicable)
   - Element sizes are consistent (if applicable)

7. Return empty containers array if no confident patterns found - let elements maintain their original hierarchy

Key changes:
- `MIN_GROUP_SIZE = 3` (was 2)
- `MIN_COVERAGE_RATIO = 0.7` (was 0.5)
- Add `isSpacingUniform(group)` helper checking max/min gap ratio <= 2
- Add `areSizesConsistent(group, direction)` helper checking size variance <= 50%
