Create `src/ui/hooks/useClaude.ts` with a single export:

**useClaude(apiKey: string | null): UseClaudeReturn**

Interface for return type:
```typescript
interface UseClaudeReturn {
  /** Trigger analysis of an image */
  analyze: (imageData: Uint8Array, mimeType: string) => Promise<void>;
  /** True while API request is in progress */
  isAnalyzing: boolean;
  /** User-friendly error message, or null */
  error: string | null;
  /** Analysis result text, or null */
  result: string | null;
  /** Clear error state */
  clearError: () => void;
  /** Clear result state */
  clearResult: () => void;
  /** Cancel any in-flight request */
  cancel: () => void;
}
```

Implementation details:

1. **State variables:**
   - `isAnalyzing: boolean` (useState, default false)
   - `error: string | null` (useState, default null)
   - `result: string | null` (useState, default null)
   - `abortControllerRef: React.MutableRefObject<AbortController | null>` (useRef)

2. **analyze function (useCallback):**
   - Check if apiKey is null: set error "API key not configured. Please add your key in Settings." and return early
   - Cancel any existing request: `abortControllerRef.current?.abort()`
   - Create new AbortController: `abortControllerRef.current = new AbortController()`
   - Set states: `setIsAnalyzing(true)`, `setError(null)`, `setResult(null)`
   - In try block:
     - Convert image: `const base64 = uint8ArrayToBase64(imageData)`
     - Create client: `const client = createClaudeClient(apiKey)`
     - Call API: `const response = await analyzeImage(client, base64, mimeType, abortControllerRef.current.signal)`
     - Set result: `setResult(response)`
   - In catch block:
     - Check AbortError: `if (err instanceof Error && err.name === 'AbortError') return;`
     - Set error: `setError(getErrorMessage(err))`
   - In finally block:
     - Set loading: `setIsAnalyzing(false)`
   - Dependencies: [apiKey]

3. **cancel function (useCallback):**
   - Call `abortControllerRef.current?.abort()`

4. **Cleanup effect (useEffect):**
   - Return cleanup function that calls `abortControllerRef.current?.abort()`
   - This prevents state updates on unmounted component
   - Empty dependency array []

5. **clearError and clearResult (useCallback):**
   - Simply set respective state to null

Import requirements:
- useState, useEffect, useCallback, useRef from 'react'
- createClaudeClient, analyzeImage, getErrorMessage from '../services/claude'
- uint8ArrayToBase64 from '../utils/base64'
