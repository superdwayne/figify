  <files>src/main/shadcn/variantResolver.ts</files>
  <action>
Create variantResolver.ts with style resolution functions:

1. resolveStyles(spec: ComponentSpec, variant: string, size: string): ResolvedStyles
   - Get variantStyles from spec.variants[variant] or spec.variants[spec.defaultVariant]
   - Get sizeStyles from spec.sizes[size] or spec.sizes[spec.defaultSize]
   - Merge in order: baseStyles -> sizeStyles -> variantStyles
   - Return ResolvedStyles object

2. mergeWithOverrides(resolved: ResolvedStyles, elementStyles: ElementStyles): ResolvedStyles
   - If elementStyles.backgroundColor differs significantly from resolved, use AI color
   - "Differs significantly" = not matching any known Shadcn color (use loose color comparison)
   - This allows custom-themed screenshots to preserve their colors
   - Default to spec colors for standard Shadcn appearances

3. colorsMatch(color1: string, color2: string, tolerance: number = 10): boolean
   - Helper to compare hex colors with tolerance
   - Convert to RGB, compare each channel within tolerance
   - Used to detect if AI color is close enough to spec default
  </action>

  <done>resolveStyles merges base+size+variant correctly, mergeWithOverrides preserves AI colors when they differ from spec</done>

Create componentFactory.ts with ShadcnComponentFactory class:

2. createComponent(element: UIElement, parent: FrameNode): SceneNode
   - Look up spec from COMPONENT_SPECS.get(element.component)
   - If no spec found, call createGenericElement() for fallback
   - Resolve styles using resolveStyles() + mergeWithOverrides()
   - Create appropriate node (frame for containers, text for Typography)
   - Apply styles using StyleApplier methods
   - Return created node

3. createGenericElement(element: UIElement, parent: FrameNode): SceneNode
   - Create frame with element.bounds
   - Apply raw element.styles directly (no spec transformation)
   - This handles unknown component types gracefully

4. applyResolvedStyles(node: FrameNode, styles: ResolvedStyles): void
   - Apply backgroundColor using styleApplier.applyFills()
   - Apply borderColor/Width using styleApplier.applyStrokes()
   - Apply borderRadius using styleApplier.applyCornerRadius()
   - Apply padding if Auto Layout is enabled
