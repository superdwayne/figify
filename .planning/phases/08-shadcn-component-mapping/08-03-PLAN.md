  <files>src/main/shadcn/variantResolver.ts</files>
  <action>
Add variant inference functions to variantResolver.ts:

1. inferButtonVariant(styles: ElementStyles): string
   - If backgroundColor is transparent/white AND no borderColor -> 'ghost'
   - If backgroundColor is transparent/white AND has borderColor -> 'outline'
   - If backgroundColor matches destructive red (#EF4444 with tolerance) -> 'destructive'
   - If backgroundColor is light gray (#F4F4F5 area) -> 'secondary'
   - If backgroundColor is dark (#18181B area) -> 'default'
   - Fallback: 'default'

2. inferBadgeVariant(styles: ElementStyles): string
   - Similar logic to button but simpler:
   - Red background -> 'destructive'
   - Light gray -> 'secondary'
   - Has border -> 'outline'
   - Fallback: 'default'

3. inferVariant(component: string, styles: ElementStyles): string | undefined
   - Switch on component type
   - Call appropriate infer function for Button, Badge
   - Return undefined for components without variant inference (Card, Input)

Helper: isTransparentOrWhite(color?: string): boolean
- Returns true for undefined, 'transparent', '#FFFFFF', '#FFF', '#ffffff'
