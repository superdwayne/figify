# Plan 07-01: Auto Layout Detection and Configuration

## Objective
Create a LayoutAnalyzer class that detects layout patterns from element positions and configures Auto Layout accordingly.

## Implementation

### 1. LayoutAnalyzer Class
Created `src/main/generator/layoutAnalyzer.ts` with:
- Layout direction detection (HORIZONTAL vs VERTICAL vs NONE)
- Position variance analysis to determine dominant direction
- Threshold-based alignment detection

### 2. Layout Detection Algorithm
- Calculate center points of all children
- Compare variance in X and Y positions
- Use range and variance ratios to determine layout direction
- Handle edge cases (single child, no clear pattern)

### 3. Integration with FigmaGenerator
- Added LayoutAnalyzer as class member
- Analyze layout before creating container frames
- Apply detected layout mode to frames

## Files Modified
- `src/main/generator/layoutAnalyzer.ts` (new)

- `src/main/generator/types.ts` (updated)

## Success Criteria
