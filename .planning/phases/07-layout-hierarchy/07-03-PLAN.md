# Plan 07-03: Spacing and Alignment Application

## Objective
Apply item spacing, padding, and alignment to Auto Layout frames based on detected patterns.

## Implementation

### 1. Item Spacing Calculation
- Calculate gaps between consecutive children
- Use median gap for resilience to outliers
- Round to nearest pixel for clean values

### 2. Padding Calculation
- Find bounding box of all children
- Calculate padding from parent bounds to children bounding box
- Handle single child case separately

### 3. Alignment Detection
- Primary axis: MIN, CENTER, MAX, or SPACE_BETWEEN
- Counter axis: MIN, CENTER, MAX
- Use spacing differential to determine alignment
- Detect space-between when equal padding on both sides

### 4. StyleApplier Enhancements
- Added `applyAutoLayout` method for comprehensive layout setup
- Added `applyItemSpacing` for gap control
- Added `applyCenteredLayout` for centered content

- `src/main/generator/layoutAnalyzer.ts`
