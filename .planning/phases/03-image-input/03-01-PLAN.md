<task type="auto">
  <name>Task 2: Create useImageCapture hook</name>
  <files>src/ui/hooks/useImageCapture.ts</files>
  <action>
Create the custom hook following the research pattern exactly. The hook must:

**Interface exports:**
```typescript
interface CapturedImage {
  blob: Blob;
  uint8Array: Uint8Array;
  previewUrl: string;
  mimeType: string;
}

interface UseImageCaptureReturn {
  capturedImage: CapturedImage | null;
  isDragging: boolean;
  error: string | null;
  clearImage: () => void;
  dropZoneProps: {
    onDragOver: (e: React.DragEvent) => void;
    onDragLeave: (e: React.DragEvent) => void;
    onDrop: (e: React.DragEvent) => void;
  };
}
```

**Implementation requirements:**

1. **State management:**
   - `capturedImage: CapturedImage | null` - the captured image data
   - `isDragging: boolean` - true when dragging over drop zone
   - `error: string | null` - validation error message

2. **processImage async function:**
   - Validate using `isValidImageType` from imageUtils
   - If invalid, set error using `getImageValidationError` and return early
   - Clean up previous previewUrl with `URL.revokeObjectURL()` if exists
   - Clear error state
   - Convert blob to Uint8Array via `blob.arrayBuffer()`
   - Create preview URL via `URL.createObjectURL(blob)`
   - Set capturedImage state

3. **Paste event handler (useEffect):**
   - Listen on `document` for 'paste' event
   - Check `e.clipboardData?.items` for image types
   - Use `item.type.startsWith('image/')` to find images
   - Call `item.getAsFile()` to get Blob
   - Call processImage with the blob
   - Return cleanup function to remove listener

4. **Window-level drag prevention (useEffect):**
   - CRITICAL: Add dragover and drop listeners to `window`
   - Both must call `e.preventDefault()` and `e.stopPropagation()`
   - This prevents browser from opening dropped files
   - Return cleanup function to remove listeners

5. **Object URL cleanup (useEffect):**
   - Return cleanup function that revokes previewUrl on unmount

6. **clearImage function:**
   - Revoke previewUrl if exists
   - Set capturedImage to null
   - Set error to null

7. **dropZoneProps object:**
   - onDragOver: preventDefault, stopPropagation, set dropEffect to 'copy', setIsDragging(true)
   - onDragLeave: preventDefault, stopPropagation, setIsDragging(false)
   - onDrop: preventDefault, stopPropagation, setIsDragging(false), find image file, call processImage

**Import from imageUtils:** `isValidImageType`, `getImageValidationError`

Follow the exact pattern from 03-RESEARCH.md "Complete useImageCapture Hook" section.
